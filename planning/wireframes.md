# Wireframes

Reference the Creating an Entity Relationship Diagram final project guide in the course portal for more information about how to complete this deliverable.

## List of Pages

- Events list (home page) ⭐
- Event detail page ⭐
- Recipe library ⭐
- User profile page
- Create/edit event form
- Create/edit recipe form

## Wireframe 1: [page title]

The home page displays a navigation bar with links to Events, Recipes, and Profile. Below is a "Create event" button and a list of event cards. Each card shows the event title, host name, date/time/location, RSVP count, dishes claimed count, and buttons to RSVP or view details.

## Wireframe 2: [page title]

Shows full event details including title, host, date/time/location, and description. The host sees Edit and Delete buttons. Below is a "Claimed dishes" section listing each claimed recipe with the claimers name. A "+ Claim a dish" button opens a modal overlay for selecting a recipe. Users who claimed a dish see an Unclaim button.


## Wireframe 3: [page title]

Displays a grid of recipe cards with image placeholder, name, category, and star rating. At the top are filter pills for each category (All, Appetizer, Main, Side, Dessert, Drink) and a sort dropdown (A-Z, Rating). Each card has Edit and Delete buttons. A "+ Add recipe" button is at the top right.
