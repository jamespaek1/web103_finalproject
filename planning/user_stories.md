# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. **Host** — a user who creates and manages potluck events, sets event details, and oversees what dishes are being brought.
2. **Guest** — a user who browses available events, RSVPs to attend, claims dishes to bring, and leaves ratings on recipes after the event.

## Draft User Stories

1. As a **host**, I want to create a new potluck event with a title, date, time, and location, so that I can invite people to my gathering.
2. As a **host**, I want to edit my event's details after creating it, so that I can update the time or location if plans change.
3. As a **host**, I want to delete an event I created, so that I can remove it if the gathering is cancelled.
4. As a **host**, I want to see a list of all dishes that have been claimed for my event, so that I can make sure there is a good variety of food.
5. As a **host**, I want to view which guests have RSVPed to my event, so that I can plan for the right number of attendees.
6. As a **guest**, I want to browse all upcoming potluck events, so that I can find gatherings I'd like to attend.
7. As a **guest**, I want to RSVP to an event, so that the host knows I plan to attend.
8. As a **guest**, I want to browse the recipe library and filter by category (appetizer, main, side, dessert, drink), so that I can quickly find the type of dish I want to bring.
9. As a **guest**, I want to claim a dish for a specific event using a pop-up modal, so that I can commit to bringing something without leaving the event page.
10. As a **guest**, I want to unclaim a dish I previously claimed, so that I can change my mind if my plans change.
11. As a **guest**, I want to add a new recipe to the shared recipe library, so that I can contribute dishes other people might want to bring to future events.
12. As a **guest**, I want to edit a recipe I added, so that I can fix any mistakes or update the details.
13. As a **guest**, I want to sort recipes by name or rating, so that I can find the most popular or easily browse alphabetically.
14. As a **host**, I want to view my profile page showing all the events I'm hosting, so that I can manage my upcoming gatherings in one place.
15. As a **guest**, I want to view my profile page showing the events I've RSVPed to and the dishes I've claimed, so that I can keep track of my commitments.
